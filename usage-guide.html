🎯 调音器

琵琶标准定弦：A₂ — D₃ — E₃ — A₃

四根弦

  • 四 缠弦 — A₂ (110.00 Hz) — 最粗
  • 三 老弦 — D₃ (146.83 Hz)
  • 二 中弦 — E₃ (164.81 Hz)
  • 一 子弦 — A₃ (220.00 Hz) — 最细

使用方法

  • 点按琴弦按钮（四/三/二/一）选择要调的音
  • 点击 🔊 播放参考音 聆听标准音高
  • 点击麦克风图标开始用麦克风实时调音
  • 扇形表盘显示音分偏差：
    🟢 绿色 = 调准 (±3¢以内)
    🟠 橙色 = 偏低（指针向左）
    🔴 红色 = 偏高（指针向右）
🔒 麦克风需要 HTTPS 安全连接。如果使用 localhost 或本地 HTTP，可通过 GitHub Pages 链接访问，或使用「播放参考音」手动调音。
💡 调音时拨弦要干脆，等余音平稳后再观察指针。尽量在安静环境中使用麦克风调音。

🥁 节拍器

BPM（速度）调节

  • 拖动滑条 — 范围 40-208 BPM
  • 刻度小圆点 — 滑条下方的 40/60/80…200 刻度，点击快速跳转
  • ＋/－按钮 — 每次增减 1 BPM
  • 👆 打拍按钮 — 手指按节奏点击，自动计算 BPM（关闭节拍器时可用）

拍号设置

  • 每小节拍数 — 1 到 16 拍
  • 单位拍 — 可选 1/2/4/8/16 分音符

节拍圆点

每个圆点代表一拍，运行时会逐拍高亮发光。点击圆点可切换模式：

  • 🔵 重音 强拍 — 实心带光泽，声音响亮（1000Hz）
  • 🟢 轻音 弱拍 — 半透明，声音柔和（600Hz）
  • ⚪ 休止 休止 — 空心虚线，静音不发声

启动 / 停止

  • 点击 ▶ 启动 开始节拍器
  • 运行中按钮变为红色 ■ 停止
  • 停止后自动记录一条练习日志

📋 预设系统

4 个默认预设涵盖琵琶基本功练习：

  • 🖐️ 轮指练习 — 60 BPM, 4/4 拍 — 轮指颗粒性训练
  • 🎯 弹挑基础 — 72 BPM, 2/4 拍 — 弹挑基本功
  • 🎸 扫弦节奏 — 88 BPM, 2/4 拍 — 扫弦节奏感
  • ⚡ 快速训练 — 132 BPM, 4/4 拍 — 提速练习

每个预设右侧的 ✎ 按钮可编辑名称、BPM、拍号和分母。可删除或重置为默认。

📈 跟练模式

适合循序渐进的练习节奏训练。点击「跟练模式」展开设置面板。

三种模式

  • 🚀 阶梯上升 — 从起始 BPM 逐步升速，到目标后保持
  • ⛰️ 起伏练习 — 到达目标后自动降速返回起点，循环反复
  • 🔥 挑战模式 — 持续升速，直到手动停止

参数设置

  • 起始 BPM / 目标 BPM — 速度范围
  • 每次 + — 每级升速幅度（1-20 BPM）
  • 每级小节 — 每个速度等级持续多少小节后升速
💡 推荐：轮指练习设起始 50、目标 100、每次 +2、每级 8 小节，从慢到稳再到快。

📊 练习日志

每次停止节拍器都会自动保存一条记录，显示在日志面板中。

  • 总练习数 — 累计练习次数
  • 连续天数🔥 — 按日期统计的连续练习天数
  • 最高 BPM🏆 — 历史最高速度记录
  • 本周次数⏱ — 最近 7 天练习次数

点击 ▶ 继续练 可一键恢复某次练习的 BPM 和拍号设置。

日志保存在浏览器本地 (localStorage)，最多 200 条。

🎨 配色主题

顶部可切换三种主题，适配不同使用场景：

  • 🟦 深蓝 — 默认主题，沉稳护眼，适合夜间练琴
  • 🪵 木纹 — 暖木色调，温和舒适
  • 🟣 紫兰迪 — 低饱和紫色，莫兰迪色系，柔和雅致

📲 安装到主屏幕

支持 PWA（渐进式 Web 应用），可像原生 App 一样安装使用：

  • Safari (iPhone/iPad): 底部分享按钮 →「添加到主屏幕」
  • Chrome (Android): 菜单 →「添加到主屏幕」
  • 桌面 Chrome: 地址栏右侧安装图标 ⊕
💡 安装后离线也可使用节拍器（调音器仍需网络加载）
