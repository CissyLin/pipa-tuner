# 🎵 琵琶调音器 — Pipa Tuner

🥚 蛋蛋Cissy联合出品 · 纯前端调音器

实时麦克风自动调音，支持四根琵琶弦（老弦 A3 / 缠弦 D4 / 中弦 E4 / 子弦 A4）。扇形调音表，播放参考音。

---

## 🚀 部署方式

### 方法一：Vercel（推荐，自动 HTTPS）

1. 打开 https://vercel.com
2. 注册/登录（可以用 GitHub 账号）
3. 点击 **Add New → Project**
4. 上传本文件夹（或连接 GitHub 仓库）
5. 保持默认设置，点 **Deploy**
6. 部署完成 → 得到一个 `xxx.vercel.app` 的链接

> 手机浏览器打开 → 添加主屏幕 → 像 App 一样用 ✅

### 方法二：个人网站

1. 将本文件夹所有文件上传到网站目录（如 `你的域名/tuner/`）
2. 用手机浏览器访问 `你的域名/tuner/`
3. 点击浏览器菜单 → **添加到主屏幕**
4. 如果没有 HTTPS 麦克风可能无法使用，需联系主机服务商开启 SSL

---

## 📱 使用说明

1. 点击麦克风按钮开启监听
2. 点击左侧琴轴按钮切换要调的弦（默认缠弦 D4）
3. 拨动琴弦，看扇形表的指针：
   - ❄️ 指到 **0** = 调准了
   - 👈 偏左 = 偏低，拧紧弦轴
   - 👉 偏右 = 偏高，放松弦轴
4. 点"参考音"播放纯正弦波标准音
