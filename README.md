# 🎵 琵琶调音器 — Pipa Tuner

🥚 蛋蛋Cissy联合出品 · 纯前端调音器

实时麦克风自动调音，支持四根琵琶弦（老弦 A3 / 缠弦 D4 / 中弦 E4 / 子弦 A4）。扇形调音表，播放参考音。

## 📱 使用说明

1. 点击麦克风按钮开启监听
2. 点击左侧琴轴按钮切换要调的弦（默认缠弦 D4）
3. 拨动琴弦，看扇形表的指针：
   - ❄️ 指到 **0** = 调准了
   - 👈 偏左 = 偏低，拧紧弦轴
   - 👉 偏右 = 偏高，放松弦轴
4. 点"参考音"播放纯正弦波标准音
